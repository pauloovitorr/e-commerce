Projeto clone do Mercado Livre

Foi desenvolvido com o objetivo de treinar React e o consumo de API, no qual foi muito proveitoso
para o meu desenvolvimento. A ideia inicial era apenas exibir os dados e coloca em um 'carrinho', porém
durante o desenvolvimento, resolvi adicionar o React para trabalhar com os componentes e estados
adicionando assim uma ótima experiência ao projeto.

Finalizado o projeto, agora é possível buscar dados digitando no campo de buscas ou mesmo por categoria,
ter o carrinho atualizado em tempo real tanto ao adicionar com excluir itens, e abrir detalhes sobre os produtos.

Projeto continuará em desenvolvimento, adicionando uma lógica para finalizar a compra, não foi criado sessões, pois 
o intuito era apenas o consumo da API. Fico à disposição em evetuais dicas ou conselhos.
